Nome do projeto : Saber+ 
Tema: Quiz, jogo educacional simples, com perguntas de conhecimentos gerais de múltipla escolha carregadas a partir de um arquivo externo 
--------
Bibliotecas utilizadas: 
Tkinter: utilizada para criação de interface gráfica 
Pandas: leitura e manipulação de arquivo Excel que continha as perguntas 
csv: para gravação de resultado de jogadores 
random math: seleção aleatória de perguntas 
datetime: registro em arquivo csv de data e hora dos resultados 
--------
Instruções de execução:
1- Garanta que o python 3 esteja instalado 
2- Certifique-se de ter todas as bibliotecas utilizadas instaladas no computador 
3- Para funcionamento do jogo, execute o arquicvo Main.py 
4- Digite seu nome e responda as perguntas apresentadas
--------
Funcionalidades:
Interface gráfica interativa
Leitura automática de perguntas a partir de arquivo Excel
Seleção aleatória de perguntas
Sistema de pontuação
